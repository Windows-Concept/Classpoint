3

Create Your Timetable

You have three options:

Option A: OCR Scanning (Recommended)

  1. Click "Create Timetable"
  2. Select "OCR Scanning" tab
  3. Drag & drop your timetable image (JPG/PNG)
  4. Wait for AI to extract your schedule
  5. Review and save

Option B: Manual Entry

  1. Click "Create Timetable"
  2. Select "Manual Entry" tab
  3. Add classes one by one with times and subjects
  4. Use the time pickers for precise scheduling
  5. Save your timetable

Option C: Demo Data (For Testing)

  1. Click "Load Demo"
  2. Instant sample timetable with realistic data
  3. Perfect for exploring features

App won't start / Windows blocks it

Click "More info" ‚Üí "Run anyway" in Windows SmartScreen. This is normal for unsigned apps.

OCR not working

Ensure you have internet connection for the first OCR scan. After that, it works offline.

Widget not showing

Check if it's hidden behind other windows. Try clicking the eye icon again to toggle.

Data not saving

Make sure ClassPoint has write permissions in its folder. Try running as administrator if needed.
